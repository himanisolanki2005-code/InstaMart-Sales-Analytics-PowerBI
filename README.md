# Insta Mart Sales Analytics Dashboard using Power BI

## Project Overview
Developed an interactive sales analytics dashboard using Power BI to analyze Insta Mart sales performance, customer trends, outlet performance, and product categories. The dashboard provides key business insights through KPIs and visualizations.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
- Power Query

## Key Features
- Performed data cleaning and transformation.
- Created data models and DAX measures.
- Designed interactive dashboards and KPI cards.
- Analyzed sales performance across outlets and city tiers.
- Evaluated product categories and customer purchasing trends.
- Implemented filters and slicers for dynamic analysis.

## KPIs Tracked
- Total Sales
- Total Orders
- Average Sales
- Average Rating
- Average Item Weight

## Project Files
- InstaMart_Sales_Dashboard.pbix

## Dashboard Insights
- Identified top-performing outlet types.
- Compared sales across different city tiers.
- Analyzed category-wise sales performance.
- Evaluated outlet size and location impact on revenue.
